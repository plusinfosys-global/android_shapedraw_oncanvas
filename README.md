# ShapeDrawOnCanvas
Capture image from Camera or select from Gallery. Edit that image by drawing different shapes on that image. 

- Camera1 api for SDKVersion < 23
- Camera2 api for SDKVersion > 23

## Shapes
- Smoothline
- Line
- Circle
- Square
- Rounded Square
